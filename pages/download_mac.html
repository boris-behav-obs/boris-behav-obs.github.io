General information

BORIS v.8 is not yet available for MacOS

A release of an old version of BORIS (v. 7.13.9) is available at https://github.com/olivierfriard/BORIS/releases/tag/v7.13.9.

Download the DMG file.

The user guide of v. 7.13.9 is available at https://boris.readthedocs.io/en/v7/

Otherwise if you want to run v.8 on your MacOS you have various possibilities:

  • Boot Camp Assistant
  • Parallels® Desktop for Mac (Take care to extract the BORIS archive in the Windows file system _ c:\Users\Bob for example _ )
  • Running GUI Linux in a virtual machine on a Mac
  • Switch to another OS

BORIS for MacOS v.7.13.9

This version includes the VLC libraries and the FFmpeg software and is available only in 64-bit version

64-bit BORIS for MacOS
