General information

BORIS v.8.24 is not yet available for MacOS

If you want to run v.8.24 on your MacOS you have various possibilities:

  • Use Virtualbox to run Linux or Windows as guest OS on your Mac (free and open-source software). A Linux virtual machine with BORIS installed is available. See Linux virtual machine
  • Use Parallels® Desktop for Mac (proprietary software) (Take care to extract the BORIS archive in the Windows file system _ c:\Users\Bob for example _ )
  • Use UTM a virtual machine host based on QEMU to run Linux or Windows as guest OS on your Mac
  • Boot Camp Assistant
  • Running GUI Linux in a virtual machine on a Mac
  • Switch to another OS
  • A docker image is available at https://hub.docker.com/r/olivierfriard/boris (see the start_macOS.command script)

See the revision history for details: https://github.com/olivierfriard/BORIS/wiki/BORIS-revision-history
