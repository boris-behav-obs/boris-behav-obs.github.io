Install BORIS v.7.11.1 for Debian

Debian 10 or newer

BORIS is available as a deb package.

Just download it BORIS deb package 64-bit v.7.11.1 and use the Software Installer

You should find the icon for starting BORIS in the Education menu, you do not need everything else.

See the revision history for details: https://github.com/olivierfriard/BORIS/wiki/BORIS-revision-history

If you want to report a bug, see the how to report a bug page.
