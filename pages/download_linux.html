Ubuntu / Debian

If you are using Ubuntu or Debian you can install BORIS using the deb package:

Download BORIS deb package 64-bit v.7.9.24

Other distributions

For the other distributions, download the following tar.gz archive and follow the instructions:

Download BORIS for GNU/Linux 64-bit v.7.9.24

BORIS for Linux 32-bit is no more available due to the fact that Qt is not available for 32 bits. If you are using Linux at 32-bit you must run BORIS from sources.

BORIS requires the installation of the VLC media player

Warning for VLC installation

If you are using Ubuntu and VLC is installed vith snap you have to uninstall it and re-install it with apt.

Verify if VLC is installed with snap, open a terminal and type:

snap list

if vlc is present in the list you have to uninstall it, open a terminal and type:

sudo snap remove vlc

You have then to re-install VLC with apt, open a terminal and type:

sudo apt install vlc

Warning for Ubuntu 20.04

If BORIS does not run it can be necessary to install the following libraries. Open a terminal and type:

sudo apt install libxcb-xinerama0 libxcb-icccm4 libxcb-image0 libxcb-keysyms1 libxcb-render-util0 libxcb-xkb1 libxkbcommon-x11-0
(your root password will be required)

and create a symbolic link to the libffi.so.7 library. Open a terminal and type:

sudo ln -s /usr/lib/x86_64-linux-gnu/libffi.so.7 /usr/lib/x86_64-linux-gnu/libffi.so.6
(your root password will be required)

To open a terminal on Ubuntu you can press ALT+F2 and type gnome-terminal

Run from sources

Requirement

The minimal version of Python for running BORIS is 3.6. Check the Python version installed on your system with:

python3 --version

If you want to generate transition flow graphs directly from BORIS you have to install the Graphviz package (not mandatory).

Install these packages following the instructions relative to your distribution:

Example for Debian based distributions like Ubuntu, Kubuntu, Xubuntu ...:

For Debian, open a terminal and type:

su (your root password will be required)
apt install vlc ffmpeg graphviz python3-pip

Attention: if VLC is installed with snap on your Linux distrabution you will have to uninstall it:

sudo snap remove vlc
(your password will be required)

For Ubuntu and Ubuntu like:

sudo apt install vlc ffmpeg graphviz python3-pip
(your password will be required)

It is recommended to create a Python virtual environment to do not interfere with the Python modules installed on your system:

pip3 -m venv boris_virtual_env

Activate the virtual environment:

source boris_virtual_env/bin/activate

Install BORIS with pip:

python3 -m pip install boris-behav-obs

Launch BORIS:

python3 -m boris
