BORIS with Ubuntu

Install BORIS (v.7.10.3) for Ubuntu

Ubuntu 20.04 or newer (20.10, 21.04)

BORIS is available as a deb package.

Just download it BORIS deb package 64-bit v.7.10.3 and use the Software Installer

You should find the icon for starting BORIS in the Education menu, you do not need everything else.

Older Ubuntu (18.04 - 19.10)

BORIS requires the installation of the VLC media player

Installing VLC Media Player

  1. If VLC is already installed on your system you have to check if it was installed with the snap utility. Open a terminal type:

    snap list

If VLC is on the list, you must remove it. Open a terminal and type:

sudo snap remove vlc
(your password will be required)

You have then to re-install VLC with apt, open a terminal and type:

sudo apt install vlc
(your password will be required)

2) If VLC is not installed on your system, just install it with the apt utility. Open a terminal type:

sudo apt install vlc
(your password will be required)

Downloading and installing BORIS

Download the tar.bz2 archive BORIS for GNU/Linux 64-bit v.7.10.3

Uncompress it and launch start_boris

BORIS for Linux 32-bit is no more available due to the fact that Qt is not available for 32 bits. If you are using Linux at 32-bit you must run BORIS from sources.

Warning for Ubuntu 20.04

If BORIS does not run it can be necessary to install the following libraries. Open a terminal and type:

sudo apt install libxcb-xinerama0 libxcb-icccm4 libxcb-image0 libxcb-keysyms1 libxcb-render-util0 libxcb-xkb1 libxkbcommon-x11-0
(your root password will be required)

and create a symbolic link to the libffi.so.7 library. Open a terminal and type:

sudo ln -s /usr/lib/x86_64-linux-gnu/libffi.so.7 /usr/lib/x86_64-linux-gnu/libffi.so.6
(your root password will be required)

To open a terminal on Ubuntu you can press ALT+F2 and type gnome-terminal

See the revision history for details: https://github.com/olivierfriard/BORIS/wiki/BORIS-revision-history
