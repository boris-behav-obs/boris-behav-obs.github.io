Install VLC Media Player

VLC is already installed

You only have to check if it was installed with the snap utility. Open a terminal type:

snap list

If VLC is on the list, you must remove it. Open a terminal and type:

sudo snap remove vlc
(your password will be required)

You have then to re-install VLC with apt, open a terminal and type:

sudo apt install vlc
(your password will be required)

If VLC is not installed

just install it with the apt utility, open a terminal and type:

sudo apt install vlc
(your password will be required)
